# FootRecon AI

An AI-powered PWA for ethical footprinting in cybersecurity.

## Setup
1. Install dependencies: `npm install`
2. Add API keys to `.env` (copy from `.env.example`).
3. Run: `npm start`
4. Install as PWA in browser.

## Features
- Passive/Active scanning with Nmap and Shodan.
- AI-driven risk prediction.
- .docx report generation.
- Ethical and legal-focused.
